MySpass.de Adblocker

Nie wieder Werbung oder Adblock Warnungen !

Mit dem MySpass Adblocker Chrome Plugin werden automatisch alle Videos von Werbung oder Adblock Warnungen befreit.

Einfach - Videos genießen

Plugin installieren, Seite ansurfen -> Videos genießen.

Nie wieder warten

Videos ansehen, ohne Unterbrechung.

Keine Warnung

Keine Warnungen mehr alle paar Minuten, nur weil man einen Adblocker benutzt.

Changelog

Version 0.0.12
  • Es wurde ein Problem behoben das bei den Videos nur ein schwarzer Bildschirm angezeigt wurde (Chrome und Firefox).
Version 0.0.5
  • Die Firefox version wurde aktualisiert, so das automatisch immer die aktuellste Version von dieser Seite heruntergeladen wird. Sollten Sie also noch eine ältere Version (<0.3) installiert haben wird ausdrücklich empfohlen die aktuelle Version zu installieren.
Version 0.0.2
  • Sollte Google Chrome das Plugin deaktiviert haben, da es nicht aus dem Webstore kommt, das Plugin einfach über diese Seite neu installieren, da die Erweiterung ab Version 0.0.2 im Webstore erhältlich ist.
Version 0.0.1
  • Initial release.
